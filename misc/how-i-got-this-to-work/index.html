      How I Got This To Work

      1 minute read

      Hello, and welcome to my brand-new, revamped website! Using GitHub Pages, I’ve been hosting a small website for the last few months. However, I’ve been wanting to start a blog. For this, I’m going to use Jekyll. Jekyll lets you make dynamic websites. It’s mainly used as a blogging tool, and Pages has internal support for it.

      This site uses a theme called Chirpy, but GitHub Pages does not have support for more than a couple themes. So this is how I made a workaround
